Desestructuracion de objetos

En JavaScript sirve para extraer valores de un objeto de manera rápida y asignarlos a variables individuales.

      const usuario = {
          nombre: "Alexandra",
          edad: 28,
          pais: "México"
      };
  
      // Desestructuración
      const { nombre, edad } = usuario;
  
      console.log(nombre); // "Alexandra"
      console.log(edad);   // 28

las variables dentro del objeto a desestructurar se pueden renombrar:

      const perro = {
        raza:'Puddle',
        color:'Marrón'
      }

      const {raza, color:tono} = perro
      console.log("Raza: ", raza, " Tono:", tono)
